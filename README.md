# Amazon Sales Analysis using SQL & Power BI

---

## Project Overview

This project focuses on analyzing Amazon sales data to extract meaningful business insights and support data-driven decision-making. The analysis was performed using MySQL, and the results were visualized through an interactive Power BI dashboard.

The goal of this project is to understand sales performance, customer behavior, and product trends.

---

## Problem Statement

Businesses often struggle to identify key factors affecting sales performance and customer purchasing behavior.  
This project aims to:

- Analyze sales data to uncover patterns and trends  
- Identify top-performing products and categories  
- Understand revenue distribution and growth trends  
- Provide actionable insights to improve business strategies  

---

## Tools & Technologies Used

- **MySQL** – Data cleaning, querying, and analysis  
- **Power BI** – Data visualization and dashboard creation  
- **SQL Concepts**:
  - Joins 
  - Aggregations  
  - Window Functions  
  - Subqueries  

---

## Analysis Performed

The following analyses were conducted using SQL:

- Total revenue and sales performance analysis  
- Customer purchasing behavior analysis  
- Product and category performance evaluation  
- Monthly and seasonal sales trend analysis  
- Identification of high-performing and low-performing products  
- Ranking of top customers using window functions  

---

## 📈 Key Business Insights

- A small percentage of products contribute to a large portion of total revenue, indicating a Pareto distribution  
- Sales show noticeable seasonal trends, suggesting the impact of time-based demand  
- Certain product categories generate high sales but relatively low profit, highlighting potential pricing inefficiencies  
- Customer purchasing patterns indicate repeat buying behavior, emphasizing the importance of customer retention strategies  
- Revenue is concentrated among top customers, suggesting the need for targeted marketing and loyalty programs  

---

## 📊 Dashboard Description

An interactive dashboard was created using Power BI to visualize key metrics and insights.

### Dashboard Features:

- 📌 KPI cards displaying total revenue and sales  
- 📈 Monthly sales trend analysis  
- 📦 Category-wise and product-wise performance  
- 👥 Customer insights and segmentation  
- 🎛️ Interactive filters for dynamic exploration  

The dashboard enables stakeholders to quickly understand business performance and make informed decisions.

---

## 🚀 Conclusion

This project demonstrates how SQL and data visualization tools can be combined to transform raw data into actionable insights. It highlights the importance of data-driven decision-making in improving sales performance and customer engagement.
